# Create a GitHub Account

## Purpose

You need a GitHub account before you can create a repository or run GitHub Actions workflows.

If you already have a GitHub account, you can skip to the next setup file after checking that:

- you can sign in successfully
- your email is verified
- you know whether two-factor authentication is already enabled on your account

Two-factor authentication is strongly recommended, but it does not need to block the start of class unless your instructor says so.

## Step 1: Open GitHub

Go to [https://github.com](https://github.com).

Click `Sign up`.

## Step 2: Create Your Personal Account

Follow the GitHub sign-up steps.

During signup, GitHub will ask for basic information such as:

- email address
- password
- username

Choose a username you are comfortable using in class.

## Step 3: Verify Your Email

This step is very important.

Without a verified email address, some basic GitHub tasks may not work correctly.

Open your inbox, find the GitHub email, and complete the verification step.

If you do not see the email:

- check spam or junk
- wait a few minutes
- ask GitHub to resend the email

## Step 4: Enable Two-Factor Authentication

GitHub strongly recommends two-factor authentication.

Use an authenticator app if possible.

When you set up two-factor authentication:

- save your recovery codes
- store them somewhere safe
- do not rely only on memory

If you are short on time during class setup, ask your instructor whether to finish this step now or right after the initial repository setup.

## Step 5: Sign In and Check Your Profile

After signup, sign in and make sure:

- your username looks correct
- your email is verified
- you can open your profile page

You do not need to customize your profile for this course.

## Common Problems

### I did not receive the verification email

Try these checks:

- confirm the email address was typed correctly
- check spam or junk folders
- wait a few minutes and resend the email

### I forgot my password

Use the `Forgot password` option on the sign-in page before class starts.

### I already have a GitHub account

That is fine.

Use your existing account if you can sign in and complete the checks in this file.

## Before You Continue

Continue only when all items below are true:

- I can sign in to GitHub
- my email is verified
- I know my GitHub username

Two-factor authentication is strongly recommended, but it does not need to block the start of class unless your instructor says so.
